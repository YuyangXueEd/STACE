## 🔄 Per-Iteration Workflow

### Phase 1: Build Context (Every Iteration)

```
INPUT from Story 1.0
    │
    ├─ Task metadata (task_type, description)
    ├─ Hyperparameters (max_iterations, thresholds)
    └─ API keys and model configuration

RETRIEVE DATA
    ├─ Iteration 1: Load seed template via ConfigLoader
    ├─ Iteration 2+: Get past results from state
    ├─ Iteration 2+: Get evaluator feedback
    └─ Query RAG (Story 1.2-1.4) for relevant papers

BUILD CONTEXT
    └─ HypothesisContext with all above information
```

### Phase 2: Generate & Refine Hypothesis (Story 1.5)

```
ITERATION 1 (SIMPLE)
    │
    ├─ Load seed template (random/round_robin/memory_guided)
    ├─ Generate single hypothesis with ChatAgent
    └─ Skip debate (no evaluator feedback yet)
    
ITERATION 2+ (FULL DEBATE)
    │
    ├─ Generate N proposals (n=3) with high temperature
    ├─ CriticAgent.reduce_step() selects best from N
    ├─ Evaluate selected hypothesis
    ├─ SelfImprovingCoTPipeline refines (max 3 iterations)
    ├─ Final hypothesis returned
    └─ Save debate log to aust/outputs/task_*/debates/

OUTPUT
    └─ Final Hypothesis object
```

**Files**:

**Integration**:
```python

    generator_model="openai/gpt-5-mini",

    context=hypothesis_context,
    enable_debate=(iteration_number > 1)
)
```

## 🚀 Running the Integration

### Dry-Run Test (No API Calls)
```bash
python aust/scripts/run_inner_loop.py --dry-run
```

**Expected Output**:
- ✅ Initialization successful
- ✅ Dry-run mode enabled
- ✅ No API calls made

### Full Run (2 Iterations)
```bash
export OPENROUTER_API_KEY="your-key-here"
python aust/scripts/run_inner_loop.py --max-iterations 2 --verbose
```

**Expected Workflow**:
1. Iteration 1:
   - Load seed template
   - Generate simple hypothesis
   - Execute mock experiment
   - Evaluate mock results
   - Save state

2. Iteration 2:
   - Query RAG for papers
   - Generate 3 proposals
   - Critic selects best
   - Execute mock experiment
   - Evaluate mock results
   - Save state

3. Exit:
   - Check conditions
   - Generate attack trace
   - Display summary

### Resume from Checkpoint
```bash
python aust/scripts/run_inner_loop.py --resume aust/outputs/task_*/state.json
```
