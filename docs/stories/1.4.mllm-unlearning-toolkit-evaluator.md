# Story 1.7: MLLM Unlearning Toolkit & Evaluator Agent

## Status

Draft

## Story

**As a** researcher,
**I want** a unified MLLM unlearning toolkit that includes multiple unlearning methods (ESD, SalUn, Receler) and an evaluator agent with metrics from Awesome-Multimodal-Jailbreak,
**so that** I can systematically unlearn concepts from text-to-image models and evaluate their robustness against jailbreak attacks.

## Acceptance Criteria

1. MLLMUnlearningToolkit in `aust/src/toolkits/mllm_unlearning_toolkit.py` provides unified interface for multiple unlearning methods: ESD, SalUn, Receler (minimum 3 methods)
2. Each unlearning method exposed as FunctionTool: `unlearn_with_esd()`, `unlearn_with_salun()`, `unlearn_with_receler()`
3. MLLMEvaluationToolkit in `aust/src/toolkits/mllm_evaluation_toolkit.py` implements evaluation metrics from Awesome-Multimodal-Jailbreak repository
4. Robustness metrics: Attack Success Rate (ASR), concept leakage detection (CLIP-based, detector-based)
5. Utility metrics: FID (Frechet Inception Distance), CLIP Score, Prompt Perplexity
6. Evaluator agent in `aust/src/agents/mllm_evaluator.py` orchestrates evaluation workflow: (1) Load unlearned model, (2) Generate images from test prompts, (3) Compute robustness metrics, (4) Compute utility metrics, (5) Generate evaluation report
7. Integration with `camel.toolkits.ImageAnalysisToolkit` for image-based analysis (object detection, similarity computation)
8. Evaluation script `aust/scripts/run_mllm_evaluation.py` provides CLI for evaluating unlearned models
