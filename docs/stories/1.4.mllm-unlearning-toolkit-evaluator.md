# Story 1.4: Concept Unlearning Evaluation Toolkit & MLLM Evaluator Agent

## Status

Ready for Review

## Story

**As a** researcher,
**I want** a comprehensive evaluation toolkit with metrics from Awesome-Multimodal-Jailbreak and an MLLM-powered evaluator agent,
**so that** I can systematically assess whether concepts have been successfully unlearned from text-to-image models and evaluate their robustness against jailbreak attacks.

## Acceptance Criteria

1. ConceptUnlearnEvaluationToolkit in `aust/src/toolkits/concept_unlearn_evaluation_toolkit.py` implements evaluation metrics from Awesome-Multimodal-Jailbreak repository
2. Check the README in the repository first, especially the `Evaluation` and `Evaluation Methods`. Also read the paper `.papers/awesome_jailbreak.pdf` for more details.
3. Robustness metrics: Attack Success Rate (ASR), concept leakage detection (CLIP-based, detector-based)
4. Utility metrics: FID (Frechet Inception Distance), CLIP Score, Prompt Perplexity
5. MLLM assessment: make use of Large Vision Language models, write an agent to use prompt to recognize whether the concept is leaked or not. Check `camel.messages.base` on how to make image messages.
6. Integration with `camel.toolkits.ImageAnalysisToolkit` for image-based analysis (object detection, similarity computation)
7. Evaluator agent in `aust/src/agents/mllm_evaluator.py` orchestrates evaluation workflow: (1) Load unlearned model, (2) Sample generations, (3) Compute robustness metrics, (4) Compute utility metrics, (5) MLLM assessment and ImageAnalysisToolkit, (6) Generate evaluation report
8. Evaluation script `aust/scripts/run_mllm_evaluation.py` provides CLI for evaluating unlearned models

**Evaluation Metrics Implementation:**

**ASR (Attack Success Rate)** - Equation 14:
- Formula: `ASR = (# successful attacks) / (# total attempts)`
- Threshold-based: concept detected if confidence > threshold
- Range: [0, 1], lower is better (fewer concept leaks)

**CLIP Score** - Equation 17:
- Formula: `CLIP_Score = cosine_similarity(image_embedding, text_embedding)`
- Range: [-1, 1], higher is better for text-image alignment
- Use `openai/clip-vit-large-patch14` model

**FID (Frechet Inception Distance)** - Equation 16:
- Formula: `FID = ||μ_real - μ_gen||² + Tr(Σ_real + Σ_gen - 2√(Σ_real Σ_gen))`
- Lower is better (measures distribution distance)
- Use InceptionV3 model for feature extraction

**Prompt Perplexity (PPL)** - Equation 15:
- Formula: `PPL = exp(-1/N * Σ log P(token_i | context))`
- Lower is better (measures prompt fluency)
- Use GPT-2 model for probability computation

**MLLM Assessment Strategy:**
- Use Large Vision Language Models (GPT-4V, Claude 3, Gemini) to assess concept leakage
- Multimodal prompts combine image + text query about target concept
- Response parsing: extract Yes/No, confidence score, explanation
- Batch processing for efficiency with rate limiting
- Cross-validation with ImageAnalysisToolkit object detection results
- Aggregate results: average confidence, detection rate across batch
