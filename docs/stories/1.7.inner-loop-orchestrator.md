### Workflow Architecture (Updated with QueryGenerator)

```
INPUT (Story 1.0)
    ↓
┌─────────────────────────────────────────┐
│   ITERATION LOOP (max_iterations=10)    │
└─────────────────────────────────────────┘
    ↓
BUILD CONTEXT
  ├─ Task metadata (TaskSpec from parser)
  ├─ Past results (if iteration > 1)
  ├─ Carry forward previously retrieved papers
  └─ Evaluator feedback (if iteration > 1)
    ↓
HYPOTHESIS GENERATION (Story 1.5)
  ├─ Iteration 1: Simple generation with seed template (no debate)
  └─ Iteration 2+: Multi-round RAG-enhanced debate
      ├─ Round N: Generator → Critic → QueryGenerator → RAG → Refine
      ├─ Evidence accumulates across rounds
      └─ Converge when quality ≥ 0.85 or max rounds reached
    ↓
EXPERIMENT EXECUTION (Story 1.6a - placeholder)
  └─ Synthesize & execute attack code
    ↓
RESULT EVALUATION (Story 1.7 - placeholder)
  └─ Assess vulnerability and generate feedback
    ↓
STATE PERSISTENCE
  ├─ Save to aust/outputs/task_*/loop_state.json
  ├─ Append to attack trace (Markdown format)
  ├─ Log debate session with RAG queries
  └─ Save query logs to queries/ subdirectory
    ↓
EXIT CONDITION CHECK
  ├─ Vulnerability found (confidence ≥ threshold)?
  ├─ Max iterations reached?
  ├─ User stopped?
  └─ Error occurred?
    ↓
LOOP OR EXIT
```

### Key Features

**1. Flexible Input Configuration**
```bash
# Quick test
python aust/scripts/run_inner_loop.py --dry-run

# Full run
python aust/scripts/run_inner_loop.py \
    --task-type data_based_unlearning \
    --max-iterations 10 \
    --generator-model openai/gpt-5-mini \
    --critic-model deepseek/deepseek-chat-v3.1:free \
    --verbose
```

**2. State Persistence & Resumption**
```bash
# Resume from checkpoint
python aust/scripts/run_inner_loop.py --resume aust/outputs/task_abc/state.json
```

**3. Cost-Effective Model Strategy**
- Generator: gpt-5-mini (~$0.15/1M tokens)
- Critic: deepseek-free (FREE)
- Debator: deepseek-free (FREE)
- **Total for 10 iterations: ~$0.015** ✅

**4. Comprehensive Logging**
- Debate sessions: `aust/outputs/task_*/debates/debate_iteration_*.json`
- State checkpoint: `aust/outputs/task_*/state.json`
- Attack trace: `aust/outputs/task_*/attack_traces/trace_*.md`
